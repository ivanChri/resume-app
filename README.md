Vue Resume-app
A client-side application inspired by the resume editor from resume.io.

- Features:
1. Real-time preview
2. Customizable themes and typography
3. Supports PDF export using html2pdf.js
4. Rich text description editor using Tiptap
5. Supports temporary data storage in localStorage using the pinia-plugin-persistedstate plugin
- tech stack
1. vue 3
2. vite
3. Tailwind CSS
4. Typescript
5. Pinia
6. Html2pdf.js
7. Pinia-plugin-persistedstate
8. Tiptap
