Vue Resume-app
A client-side application inspired by the resume editor from resume.io.

- Features:
1. Real-time preview
2. Customizable themes and typography
3. Supports PDF export using browser window.print
4. Rich text description editor using Tiptap
5. Supports temporary data storage in localStorage using the pinia-plugin-persistedstate plugin

- tech stack
1. vue 3
2. vite
3. Tailwind CSS
4. Typescript
5. Pinia
6. Pinia-plugin-persistedstate
7. Tiptap

Note: If you have layout problems with the themes when printing or exporting as a PDF, please change the margins settings from default to none or minimum in the browser's print window settings.
